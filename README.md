# Desiree Goldey — Interactive Digital CV

A personal, interactive digital CV built as a single-page website. Designed to bring a career story to life with editorial aesthetics, smooth animations, and a few surprises.

## ✨ Features

- **Editorial design** — Black and warm amber palette with Playfair Display typography
- **Animated stat counters** — Key impact numbers roll up on load
- **Scroll interactions** — Custom amber cursor and hover micro-interactions
- **Awards ticker** — Continuously scrolling recognition banner
- **🎙 Easter egg** — A hidden Spotify podcast player for [Talentless: A Recruiters' Podcast](https://www.talentlesspodcast.com/)

## 🛠 Built With

- Pure HTML, CSS, and vanilla JavaScript — zero frameworks, zero dependencies
- Google Fonts (Playfair Display, DM Mono, Libre Baskerville)
- Spotify Embed API
